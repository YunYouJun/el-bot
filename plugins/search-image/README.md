# 以图搜图

> 使用 [SauceNAO](https://saucenao.com/) 搜图。

请前往 [search-api | SauceNAO](https://saucenao.com/user.php?page=search-api) 申请 api key。

```sh
yarn add sagiri
```

## 配置

```yaml
search-image:
  token: xxx
  options:
    # 返回多少个结果
    results: 3
```

More options in [Sagiri](https://github.com/ClarityCafe/Sagiri).
